## ⚙️ Kullanılan Teknolojiler
- **Python 3.x**
- **Tkinter** – Grafiksel kullanıcı arayüzü (GUI) için  
- **zipfile** / **zlib** – Dosya sıkıştırma işlemleri  
- **os** – Dosya yolu işlemleri
